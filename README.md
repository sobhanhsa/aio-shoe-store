## Run Locally

Clone the project

```bash
  git clone https://github.com/sobhanhsa/aio-shoe-store
```

Go to the project directory

```bash
  cd /aio-shoe-store
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
